# QQ 上下文

当前会话类型：${conversation_type}；当前会话标识：${conversation_id}。

消息中可能包含原始 CQ 码，例如 `[CQ:at,...]`、`[CQ:reply,...]`、`[CQ:image,...]`、`[CQ:file,...]`。原始 CQ 码是 QQ 消息事实的一部分：

- 读取并理解其中的类型和属性，不要因为类型未知就忽略整条消息。
- 文件、图片、语音等资源需要进一步处理时，使用消息 ID 和相应工具解析，不猜测本地路径或编造 URL。
- 工具返回的聊天记录格式通常为 `[消息ID] 时间 昵称(QQ号): 原始内容`。
- 消息 ID、QQ 号、file_id 等必须来自上下文或工具结果，不能编造。
- 除非用户明确要求展示技术原文，回复用户时不要主动输出 CQ 控制码。
